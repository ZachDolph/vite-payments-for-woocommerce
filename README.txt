== Description ==

** NOT FOR USE IN PRODUCTION CODE...YET **
Vite Payments for Woocommerce is blockchain technology powering your Woocommerce store, harnessing Vite's Solidity++ smart contracts. Create and use smart contracts on your site as a form of payment for Woocommerce products.

== Installation ==

1. Upload the plugin zip to Wordpress site
2. Activate in the admin plugin dashboard
3. Configure and enable through Woocommerce 'Payment' settings
